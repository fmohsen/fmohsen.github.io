UROP PROJECT #UR329

An Android application is composed of one or more of the following components: activity, service, content provider, and broadcast receiver. The Broadcast receivers are used as a mean of communication between app’s components, between different apps, and between the OS and apps. The later enables applications to register for system events or actions (e.g. receive call, receive message). Once an event occurs, Android run-time notifies all applications.
